Please consider putting your files into the following directories:
  • "src/app/components"
    components directory
  • "src/app/consts"
    constants directory
  • "src/app/entities"
    entities directory, eg models for API, external responses, etc
  • "src/app/styles"
    a common directory for global app styles; by default, it contains default custom styles
All the given directories needs to be used as @app/consts/lang.const or @app/components/button.component, for instance.
Necessary settings are already added to the tsconfig file for it.
